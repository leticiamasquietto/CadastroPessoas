# CadastroPessoas

Este projeto foi desenvolvido para a disciplina de Desenvolvimento de Aplicações Mobile I, com o objetivo de aplicar, na prática, os conceitos estudados ao longo da matéria.

## Sobre o Sistema

A aplicação consiste em um aplicativo Android desenvolvido com java que realiza a coleta de dados na tela inicial e um processamento básico desses dados na segunda tela, além de uma tela de informações com dados sobre as desenvolvedoras e a matéria.

## Funcionalidades

O sistema implementa coleta de dados na tela principal, que envia essas informações para a segunda tela através de um intent.

Na tela de confirmação de informações, os dados preenchidos na tela inicial são apresentados novamente, e o usuário confirma se estão corretos. Se sim, a tela de confirmação fecha, voltando para a inicial, e um toaster confirmando que os dados foram processados aparece na tela. Se não, a tela de confirmação fecha, voltando para a inicial, e os dados estão disponíveis para serem editados.

Existe também uma tela de informações do projeto que é acessada através de um menu na tela principal, onde aparecem as informações de:
- Título do trabalho
- Nome da disciplina
- Nome e RA das desenvolvedoras

## Interface

A interface foi desenvolvida usando xml e ícones do Material Design, focando na organização visual e na simplicidade.
